GIF Previews for Products Section
==================================

Add GIF previews for each product. Recommended size: 400x300px or similar aspect ratio.
These will appear on hover in the "Products Built With This System" section.

Required files:
- landing-page.gif
- realgreatdevs.gif
- studio.gif
- businessos.gif
- pivotal.gif
- raise.gif
- revivedeadlinks.gif
- devthusiast.gif
- qwiklocker.gif

Tips:
- Use screen recordings converted to GIF
- Keep file sizes under 2MB for performance
- 5-10 second loops work best
